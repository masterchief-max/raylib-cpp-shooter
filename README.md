A basic space ship shooter just like the good old space invaders but now enemy has different moods(It is still in prototype stage and has lot to come up)


Raylib-Cpp-Shooter

## Building
1. git clone https://github.com/masterchief-max/raylib-cpp-shooter
2. cd raylib-cpp-shooter
3. make
4. Or use Vs code and copy src and run with a template or in your own setting,ide,enviroment etc.

## objectives:
kill the enemy ship

## controls:
1. WASD move
2. SPACE to shoot
3. R to restart

## features:
1. enemy has moods which change according to health status.


## purpose:
1. I built this game as a commitment to building stuff and learning C++ and becoming a better programmer.
2. This game can be used for learning or as an inspiration.
